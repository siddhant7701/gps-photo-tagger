# 📍 GPS Photo Tagger

A simple and powerful web app that allows users to upload images and add real-time GPS-style overlays including location details, coordinates, altitude, and timestamp — similar to GPS camera apps.

---

## 🚀 Live Demo
👉 https://siddhant7701.github.io/gps-photo-tagger/

---

## ✨ Features

- 📸 Upload any image
- 📍 One-click location presets
- 🧭 Displays:
  - Address
  - Latitude & Longitude
  - Altitude
- ⏱️ Auto timestamp (Local + GMT)
- 🖼️ Clean GPS-style overlay UI
- 🏷️ Custom logo + branding ("GPS Map Camera")
- ⬇️ Download final image
- 📱 Mobile-friendly design

---

## 🛠️ Tech Stack

- HTML5
- CSS3

## ⚙️ How to Use

1. Upload an image
2. Select a location preset
3. Click **Generate Image**
4. Click **Download**
